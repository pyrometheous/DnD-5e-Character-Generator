# GUI Assets And Credits

## Expected Asset Directory

The GUI expects this folder in project root:

- `GUI/`

And these files inside it:

- `GUI/parchment_bg.jpg`
- `GUI/Cinzel-Regular.ttf`
- optional: `GUI/ASSET_SOURCES.md`

`main.pyw` currently points to this exact location via:

- `GUI_DIR = BASE_DIR / "GUI"`
