## Notes

- The packaged GUI theme intentionally avoids generative-AI-created artwork.
- Keep this file with the project so the provenance of bundled GUI assets stays clear.
- For spell cards, compact icon badges are used in subtitles for school, ritual, and concentration.
- The margin key is rendered per page and only includes icon entries that appear on that page.
- The Bootstrap circled-C icon is used for concentration; school/ritual badges are rendered programmatically from abbreviations.
